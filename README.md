# Doca

Public CDN for Doca app auto-updates.

**Doca is a paid markdown editor — purchase at [doca.md](https://doca.md).**

This repository hosts release binaries so that licensed copies of Doca can
auto-update via electron-updater. The binaries here are governed by the
LICENSE included in this repo — redistribution is prohibited.

## Where to buy

- Website: [doca.md](https://doca.md)
- Direct: link available after purchase

## What's here

- `.dmg` — installer for new users (linked from your purchase email)
- `.zip` — auto-updater payload (read by existing installs)
- `latest-mac.yml` — update manifest
- `.blockmap` — differential update metadata

## What's NOT here

- Source code (private)
- Issue tracker — for support, email [hello@doca.md](mailto:hello@doca.md)
- Wiki — see [doca.md](https://doca.md)
